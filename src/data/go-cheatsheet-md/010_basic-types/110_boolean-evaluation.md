## タイトル
title: 真偽値になる式 (比較演算)

## タグ
tags: ["basic-types", "真偽値", "bool", "比較演算子", "if"]

## コード
```go
package main

import "fmt"

func main() {
	age := 25
	score := 85
	name := "Alice"

	if age >= 20 {
		fmt.Println("成人です。")
	}

	if name == "Alice" {
		fmt.Println("こんにちは、Alice さん！")
	}

	// 複数の条件を && (AND) で組み合わせる
	if age >= 20 && score >= 70 {
		fmt.Println("あなたは条件を満たしています。")
	}
}
```

## 解説
```text
`if` 文などの条件分岐では、`bool` 型の値だけでなく、
評価結果が `true` または `false` になる**式**を
条件としてよく使います。代表例が**比較演算子**です。

**比較演算子:**
二つの値を比較し、結果を `bool` 型で返します。
*   `==` : 等しい
*   `!=` : 等しくない
*   `<`  : より小さい
*   `<=` : より小さいまたは等しい
*   `>`  : より大きい
*   `>=` : より大きいまたは等しい

これらは数値や文字列などで使用できます。

**`if` 文での利用:**
比較演算子の結果は `bool` なので、
そのまま `if` 文の条件に使えます。
例: `if age >= 20 { ... }`
    `if name == "Alice" { ... }`

**論理演算子との組み合わせ:**
比較演算の結果を `&&` (AND) や `||` (OR) で
組み合わせることで、より複雑な条件を表現できます。
例: `if age >= 20 && score >= 70 { ... }`
    (年齢が20以上 **かつ** スコアが70以上)
