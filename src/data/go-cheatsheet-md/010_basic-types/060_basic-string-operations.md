## タイトル
title: 文字列の基本操作 (長さ, アクセス, 部分文字列)

## タグ
tags: ["basic-types", "文字列", "string", "len", "インデックス", "スライス"]

## コード
```go
package main

import "fmt"

func main() {
	s := "abcdefg"

	sub1 := s[1:3] // "bc"
	fmt.Printf("s[1:3] = \"%s\"\n", sub1)

	sub2 := s[:4] // "abcd"
	fmt.Printf("s[:4]  = \"%s\"\n", sub2)

	sub3 := s[2:] // "cdefg"
	fmt.Printf("s[2:]  = \"%s\"\n", sub3)

	sub4 := s[:] // "abcdefg"
	fmt.Printf("s[:]   = \"%s\"\n", sub4)

	fmt.Printf("元の文字列 s = \"%s\"\n", s) // 元は不変
}
```

## 解説
```text
Goの文字列 (`string`) に関する基本的な操作です。

**文字列の長さ: `len(s)`**
組み込み関数 `len()` は文字列 `s` の
**バイト数**を返します。
**注意:** UTF-8エンコーディングのため、
日本語などのマルチバイト文字が含まれる場合、
**文字数とは一致しません**。
(例: `"こんにちは"` は5文字だが `len()` は15)
文字数を数えるには `unicode/utf8` パッケージの
`RuneCountInString()` を使います。

**バイトへのアクセス: `s[i]`**
インデックス `i` (0始まり) を指定すると、
その位置の**バイト (`byte`)** にアクセスできます。
**注意:** これは**文字**へのアクセスではありません。
マルチバイト文字の場合、文字の一部バイトを返します。
また、文字列は不変なので `s[i] = 'a'` のような
代入はできません。

**部分文字列の取得 (スライス): `s[i:j]`**
範囲 `[i:j]` を指定すると、インデックス `i` から
`j-1` までのバイトシーケンスからなる
**新しい部分文字列**を取得できます。
*   `i`: 開始インデックス (含まれる)。省略すると `0`。
*   `j`: 終了インデックス (含まれない)。省略すると最後まで。
*   例: `s[1:3]` は `s[1]` と `s[2]` を含む。
*   `s[:4]` は `s[0]` から `s[3]` まで。
*   `s[2:]` は `s[2]` から最後まで。
*   `s[:]` は文字列全体のコピー。
*   元の文字列 `s` は変更されません (不変性)。
