## タイトル
title: 整数 (Integer)

## タグ
tags: ["basic-types", "整数", "int", "rune"]

## コード
```go
package main

import "fmt"

func main() {
	var i int = -100
	fmt.Printf("int 型変数 i: 値=%d\n", i)

	var r rune = 'あ'
	fmt.Printf("rune 型変数 r: 値=%c (数値=%d)\n", r, r)

	var r2 rune = 12354 // 'あ' の Unicode コードポイント
	fmt.Printf("rune 型変数 r2: 値=%c\n", r2)
}

```

## 解説
```text
**整数**は小数点を含まない数値です。
Goには、扱う数値の大きさや符号の有無に応じて
様々な整数型があります。

**符号付き整数:**
マイナス、0、プラスの値を扱えます。
*   `int`: サイズが実行環境に依存
    (32bit環境で`int32`, 64bit環境で`int64`)。
    **通常の整数計算には `int` を使うのが一般的**です。
*   `int8`, `int16`, `int32`, `int64`:
    サイズが固定 (8, 16, 32, 64ビット)。
    メモリ効率や特定サイズが必要な場合に使います。
    扱える範囲はビット数で決まります
    (例: `int8` は -128 〜 127)。
    巨大な値には `int64` を使います。

**`rune` 型:**
`int32` の**別名 (エイリアス)** です。
Unicode のコードポイント (文字を表す数値) を
表現するためによく使われます。
シングルクォート `'` で文字を囲むと
`rune` リテラルになります (例: `'a'`, `'あ'`)。
`rune` は実質 `int32` なので、
その範囲の数値を直接代入することも可能です。
