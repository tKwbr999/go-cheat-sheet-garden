---
title: "Slice Indexing" # タイトル内のダブルクォートをエスケープ
tags: ["data-structures"]
---

```go
// スライスのインデックス
full := []int{1, 2, 3, 4, 5}
// [2 3 4]   - インデックス 1 から 3 まで (4 は含まない)
s1 := full[1:4]
// [1 2 3]   - 開始からインデックス 2 まで
s2 := full[:3]
// [3 4 5]   - インデックス 2 から最後まで
s3 := full[2:]
// [1 2 3 4 5] - スライス全体
s4 := full[:]
```
