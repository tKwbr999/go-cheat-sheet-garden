## タイトル
title: 多次元配列 (Multi-dimensional Array)

## タグ
tags: ["data-structures", "配列", "array", "多次元配列", "2次元配列", "行列"]

## コード
```go
package main

import "fmt"

func main() {
	// 2行3列の配列をリテラルで初期化
	matrix := [2][3]int{
		{10, 20, 30}, // 行 0
		{40, 50, 60}, // 行 1
	}

	// 要素の読み取り (行インデックス, 列インデックス)
	val1 := matrix[0][1] // 1行目(0), 2列目(1) -> 20
	val2 := matrix[1][2] // 2行目(1), 3列目(2) -> 60
	fmt.Printf("matrix[0][1]=%d, matrix[1][2]=%d\n", val1, val2)

	// 要素への書き込み (代入)
	matrix[0][0] = 11
	fmt.Println("変更後:", matrix) // [[11 20 30] [40 50 60]]

	// ネストしたループで全要素アクセス
	fmt.Println("--- 全要素 ---")
	for i := 0; i < len(matrix); i++ { // 行ループ (len=2)
		for j := 0; j < len(matrix[i]); j++ { // 列ループ (len=3)
			fmt.Printf(" matrix[%d][%d]=%d\n", i, j, matrix[i][j])
		}
	}
	// 範囲外アクセスは panic (例: matrix[2][0])
}

```

## 解説
```text
Goでは、配列の要素としてさらに配列を持つ
**多次元配列**を作成できます。
よく使われるのは**2次元配列**（配列の配列）で、
表や行列のような格子状データを扱います。

**宣言/初期化:**
`var 変数名 [行数][列数]型`
`変数名 := [行数][列数]型{ {行1要素...}, {行2要素...} }`

*   `[行数][列数]` でサイズを指定します。
*   リテラル初期化では、外側の `{}` 内に、
    内側の配列（各行）のリテラルを `,` 区切りで記述します。
*   宣言のみの場合、全要素がゼロ値で初期化されます。
    `var matrix [2][3]int` // -> [[0 0 0] [0 0 0]]

**要素へのアクセス:**
2つのインデックスを使います。
`変数名[行インデックス][列インデックス]`
インデックスはそれぞれ `0` から `サイズ - 1` までです。

コード例では `matrix[0][1]` で1行目・2列目の `20` に、
`matrix[1][2]` で2行目・3列目の `60` にアクセスしています。
`matrix[0][0] = 11` のように代入も可能です。

ネストした `for` ループを使うと、全要素にアクセスできます。
外側のループが行 (`i`)、内側のループが列 (`j`) を回ります。
`len(matrix)` で行数、`len(matrix[i])` で i 行目の列数を取得できます。

**注意:** 1次元配列と同様、範囲外のインデックスで
アクセスすると `panic` が発生します。

多次元配列も固定長です。可変長の表データなどには
多次元スライス（スライスのスライス）が使われます。
