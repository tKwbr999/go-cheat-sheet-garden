## タイトル
title: 制御構文: 繰り返し処理 `for` (基本形)

## タグ
tags: ["flow-control", "for", "ループ", "繰り返し"]

## コード
```go
package main

import "fmt"

func main() {
	fmt.Println("--- 0から4までカウントアップ ---")
	for i := 0; i < 5; i++ {
		fmt.Println("現在の i:", i)
	}
	// fmt.Println(i) // エラー: i はループの外では未定義
}

```

## 解説
```text
同じ処理を繰り返すには**ループ**構文を使います。
Goのループ構文は **`for`** 文のみですが、
様々な書き方ができます。

**基本構文 (3つの部分からなる形式):**
`for 初期化ステートメント; 条件式; 後処理ステートメント { ... }`

1.  **初期化:** ループ開始前に一度だけ実行。
    通常ループカウンターを初期化 (例: `i := 0`)。
    ここで宣言された変数のスコープはループ内のみ。
2.  **条件式:** 各反復の開始前に評価。
    `true` ならループ本体を実行、`false` ならループ終了。
    (例: `i < 5`)
3.  **後処理:** ループ本体の実行後に毎回実行。
    通常カウンターを更新 (例: `i++`)。

**コード例:** `for i := 0; i < 5; i++`
*   `i := 0`: `i` を 0 で初期化。
*   `i < 5`: `i` が 5 より小さい間繰り返す。
*   `i++`: 各反復後に `i` を 1 増やす。
結果として 0 から 4 までが出力されます。

**Goの特徴:**
*   `for` の3つの部分を囲む `()` は不要。
*   ループ本体を囲む `{}` は必須。
*   `for` と `{` は同じ行に書く。

後処理ステートメントを変えることで、
カウントダウン (`j--`) やステップ変更 (`k += 2`) も可能です。
この形式は決まった回数繰り返す場合に便利です。
