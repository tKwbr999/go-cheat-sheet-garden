## タイトル
title: 関数: 可変長引数 (...) の定義

## タグ
tags: ["functions", "func", "引数", "パラメータ", "可変長引数", "...", "スライス"]

## コード
```go
package main

import "fmt"

// 任意の個数の int 引数を受け取り合計を返す
// nums は関数内で []int スライスとして扱われる
func sum(nums ...int) int {
	fmt.Printf("引数: %v (%T)\n", nums, nums)
	total := 0
	for _, num := range nums {
		total += num
	}
	return total
}

func main() {
	fmt.Println(sum(1, 2, 3))       // 引数 3つ -> 6
	fmt.Println(sum(10, 20))        // 引数 2つ -> 30
	fmt.Println(sum())              // 引数 0個 -> 0
	fmt.Println(sum(5, -1, 4, 10)) // 引数 4つ -> 18
}

```

## 解説
```text
関数の引数の個数を呼び出し時に柔軟に変えたい場合、
**可変長引数 (Variadic Arguments)** を使います。
特定の型の引数を **0個以上、任意の個数**
受け取れるように定義できます。

**定義方法: `...型`**
関数定義の**最後の引数**の型の前に `...` を付けます。
`func 関数名(..., 可変引数名 ...型)`
可変長引数は引数リストの最後に一つだけ指定可能です。

**関数内部での扱い: スライス**
関数内では、可変長引数として渡された値は、
その型 (例: `int`) の**スライス** (例: `[]int`) として
扱われます。
そのため、`len()` で個数を取得したり、
`for range` で各要素にアクセスしたりできます。

コード例の `sum(nums ...int)` では、`int` 型の引数を
0個以上受け取ります。関数内では `nums` は `[]int` 型の
スライスとして扱われ、`for range` で合計を計算しています。
`main` 関数からの様々な個数の引数での呼び出しが可能です
(引数なしの `sum()` も可)。

`fmt.Println` などもこの仕組みを利用しています。
